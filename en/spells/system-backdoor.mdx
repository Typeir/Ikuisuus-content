# System Backdoor (UA)

You create a magical backdoor in a computer system, allowing you to access and control it remotely.

---

> **System Backdoor (UA)**
> _4th-level Transmutation T_
> **Casting Time**: 1 Minute
> **Range**: Self
> **Components**: V, S, M
> **Duration**: Concentration, up to 1 hour
>
> You touch a computer system and create a magical backdoor that allows you to access and control the system remotely for the duration. While the spell lasts, you can issue commands to the system as if you were physically present, bypassing any security measures. The backdoor remains open for the duration, after which it closes and the system returns to normal.
>
> **At Higher Levels.** When you cast this spell using a spell slot of 5th level or higher, the duration increases by 1 hour for each slot level above 4th.
