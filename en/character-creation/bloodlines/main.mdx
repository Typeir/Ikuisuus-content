# Bloodlines

**Bloodlines** define ancestry in Damocles. Each bloodline has:
- **Core Features**: Always on; not priced; set identity (e.g., creature type, ASIs).
- **Boons (BP)**: A menu of purchasable traits. Costs are shown as **(X BP)** or in option tables.

**Boon Points (BP)** represent how much customization a character can take from their bloodline’s menu.

---

## Using BP

1. **Check the bloodline’s Budget.**  
   Each bloodline lists a **Baseline Budget** (e.g., **10 BP** for **Silent One**). Do not exceed that total when selecting options.

2. **Pick from Suites and Features.**  
   - Some sections are **Suites** with a table. Each table will specify whether it is multi-choice or not.
   - Other entries are standalone features with a fixed BP cost.
   - Some features may be specifically incompatible with others; read carefully.

3. **Add the costs.**  
   Your chosen options’ BP must total **the bloodline’s Budget**. Unspent BP are lost unless a rule says otherwise.
