# Using Average Values for Dice Rolls

## Overview

This rule describes using fixed, average values in place of rolling dice when a feature, spell, or effect would otherwise require rolling dice to determine hit points restored or similar effects.

## Mechanics

- When a feature or effect restores hit points by rolling dice, the DM or rules text may allow using a predetermined average value instead of rolling the dice.
- Apply the average value for each die as specified by the feature's guidance.

For example, in the source text fragment: "each)die.)For)example,)instead)of)restoring)2d6)hit) points)to)a)creature,)you)restore)12." This illustrates replacing the dice roll with a fixed total restoration rather than rolling the dice.

## Interactions

- This substitution replaces rolling the dice for the listed restoration effect only; it does not alter other effects that depend on the randomness of a roll unless the feature explicitly permits doing so.
- Use of average values should be applied consistently according to the feature or DM ruling to avoid altering expected balance unintentionally.
