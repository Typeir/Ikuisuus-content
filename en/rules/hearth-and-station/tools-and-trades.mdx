# Tools and Trades

## Overview

This page covers artisan's tools, kits, musical instruments, and the rules for tool proficiency. Tools represent specialized gear and the knowledge to use it — they are the primary instruments of craft, trade, and professional life. For the rules on crafting goods with these tools, see [Crafting and Creation](/library/rules/hearth-and-station/crafting-and-creation).

## Mechanics

### What a tool represents

A tool represents specialized gear and the knowledge required to use it. Proficiency with a tool allows you to add your proficiency bonus to ability checks you make using that tool. Tool use is not tied to a single ability: the GM determines which ability applies based on the task (for example, a Dexterity check to carve fine detail with woodcarver's tools or a Strength check to shape iron with smith's tools).

Proficiency with a set of artisan's tools lets you add your proficiency bonus to any ability check you make using those tools in pursuit of that craft. Each type of artisan's tools requires its own proficiency.

Example uses:

- Using woodcarver's tools to carve fine detail: Dexterity check.
- Using smith's tools to fashion an iron pot: Strength check (or another ability the GM deems appropriate).
- Using thieves' tools to pick a lock: Dexterity check with your proficiency bonus if proficient.

### Tools and checks — guidance for GMs

- The GM decides which ability applies to a tool-based task. Treat tools as granting specialized competence in tasks associated with that tool.
- If a task closely parallels a skill check, use the same ability but allow the tool proficiency to apply instead of (or in addition to) skill proficiency only when appropriate.
- Tool proficiency does not automatically grant training in every task associated with an item category; proficiency represents the ability to add the proficiency bonus to checks that involve the tool directly.
- Multiple proficiencies stack only where the rules explicitly allow; do not add proficiency bonus twice for the same check.

## Interactions

- Tool proficiency interacts with ability checks: add your proficiency bonus to checks made using the tool when you are proficient with it.
- Starting equipment selection: when a class grants a pack, that pack's listed items are your starting gear; buying a pack outright costs the listed gp and provides the listed contents.
- Vehicles and mounts: see the "Mounts and Vehicles" section for vehicle tools and rules.
- Crafting: see [Crafting and Creation](/library/rules/hearth-and-station/crafting-and-creation) for rules on producing goods with artisan's tools during downtime.
