# Encumbrance and Carrying Capacity

Carrying capacity in this system follows the granular 3.5-edition model. Three load tiers — light, medium, and heavy — determine speed, skill penalties, and the maximum Dexterity bonus that armor can transmit.

## Load Penalties

| Load   | Max Dex Bonus | Check Penalty | Speed Reduction | Run Multiplier |
| ------ | ------------- | ------------- | --------------- | -------------- |
| Light  | —             | 0             | None            | ×4             |
| Medium | +3            | −3            | 30 ft → 20 ft   | ×4             |
| Heavy  | +1            | −6            | 30 ft → 20 ft   | ×3             |

Speed reduction follows the same scale at other base speeds: 20 ft drops to 15 ft, 40 ft drops to 30 ft, and so on.

## Lifting and Dragging

- **Lift over head**: a creature can lift a weight up to its **Heavy Load maximum** over its head. Carrying this much treats the creature as having a heavy load.
- **Lift off the ground**: a creature can lift a weight up to **twice its Heavy Load maximum** off the ground. While doing so, the creature can only stagger, moving 5 feet per round.
- **Push or drag**: on a smooth surface, a creature can push or drag up to **five times its Heavy Load maximum**. Unfavourable conditions can halve this; favourable conditions can double it.

## Size Multipliers

Apply these multipliers to every threshold in the Carrying Capacity by Strength table, based on creature size:

| Size       | Multiplier |
| ---------- | ---------- |
| Fine       | ×1/8       |
| Diminutive | ×1/4       |
| Tiny       | ×1/2       |
| Small      | ×3/4       |
| Medium     | ×1         |
| Large      | ×2         |
| Huge       | ×4         |
| Gargantuan | ×8         |
| Colossal   | ×16        |

## Quadruped Multipliers

A quadruped (or other creature with a markedly load-bearing frame) replaces its size multiplier with the quadruped value at Medium and larger sizes:

| Size       | Quadruped Multiplier |
| ---------- | -------------------- |
| Medium     | ×1.5                 |
| Large      | ×3                   |
| Huge       | ×6                   |
| Gargantuan | ×12                  |
| Colossal   | ×24                  |

Sizes smaller than Medium continue to use the bipedal size multipliers from the previous table.

## Tremendous Strength

For Strength scores above 29, find the score in the 20–29 range that shares the same ones digit, then multiply every threshold by 4 for each full +10 above that score. Apply size and quadruped multipliers afterward.

For example, a Strength 32 creature uses Strength 22's row (173 / 346 / 520) multiplied by 4: 692 / 1384 / 2080. A Strength 42 creature uses the same row multiplied by 16.
